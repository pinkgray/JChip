# JChip
### 에자일 개발론을 활용한 프로젝트 관리 GUI 프로그램   
  
## 역할 분담
김규형, 최인효 - 로그인 및 사용자 관리, 디자인  
우리나, 정민지 - 프로젝트 관리   
문지원, 송낙규 - 프로젝트 할일 목록 관리  

## 개발 환경
- Language : Java    
- IDE : Eclipse  

## 프로젝트 정보
- DB를 사용하지 않기 때문에 해당 .dat 파일을 활용하여 데이터 관리
